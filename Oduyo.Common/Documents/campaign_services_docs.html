🎯 ICampaignService

💡 Kampanya Nedir? Belirli bir tarih aralığında paket ve modüller üzerinde indirim sağlayan promosyonlardır. Kampanyalar miktar veya oran bazlı indirim sunabilir.

Kampanya Oluşturma - CreateCampaignAsync

Web API Controller

[HttpPost]
public async Task<IActionResult> CreateCampaign([FromBody] CreateCampaignDto dto)
{
    var currentUserId = int.Parse(User.FindFirst(ClaimTypes.NameIdentifier).Value);
    
    var campaign = await _campaignService.CreateCampaignAsync(dto, currentUserId);
    return Ok(new { message = "Kampanya oluşturuldu", data = campaign });
}

3. Teklif Hazırlarken Kampanya Kontrolü

// Müşteriye teklif hazırlarken geçerli kampanyaları kontrol et

var packageId = 2; // Standard Paket
var originalPrice = 999.99m;

// Geçerli kampanyaları getir
var currentCampaigns = await _campaignService.GetCurrentCampaignsAsync();

decimal finalPrice = originalPrice;
string appliedCampaign = null;

foreach (var campaign in currentCampaigns)
{
    // Bu paket kampanyada var mı?
    var isInCampaign = await _campaignPackageService.IsPackageInCampaignAsync(
        campaign.Id, packageId);
    
    if (isInCampaign)
    {
        // İndirim uygula
        if (campaign.DiscountRate.HasValue)
        {
            finalPrice = originalPrice * (1 - campaign.DiscountRate.Value / 100);
        }
        else if (campaign.DiscountAmount.HasValue)
        {
            finalPrice = originalPrice - campaign.DiscountAmount.Value;
        }
        
        appliedCampaign = campaign.Name;
        break; // İlk bulunan kampanyayı uygula
    }
}

Console.WriteLine($"Orijinal Fiyat: ₺{originalPrice}");
if (appliedCampaign != null)
{
    Console.WriteLine($"Kampanya: {appliedCampaign}");
    Console.WriteLine($"İndirimli Fiyat: ₺{finalPrice}");
}

        lstAvailablePackages.DisplayMember = "Name";
        lstAvailablePackages.ValueMember = "Id";

        {

        }

🔒 Güvenlik ve En İyi Pratikler

1. Tarih Kontrolü

Kampanya oluştururken başlangıç tarihi bitiş tarihinden önce olmalıdır:

if (dto.StartDate >= dto.EndDate)
{
    throw new InvalidOperationException("Başlangıç tarihi bitiş tarihinden önce olmalıdır");
}

2. İndirim Kontrolü

Kampanyada hem miktar hem de oran bazlı indirim aynı anda olmamalı:

if (dto.DiscountAmount.HasValue && dto.DiscountRate.HasValue)
{
    throw new InvalidOperationException("Miktar veya oran indirimi seçilmelidir, ikisi birden olamaz");
}

if (!dto.DiscountAmount.HasValue && !dto.DiscountRate.HasValue)
{
    throw new InvalidOperationException("Kampanya için indirim tanımlanmalıdır");
}

3. Çakışan Kampanyalar

Aynı paket üzerinde çakışan kampanyalar olmamalı veya öncelik sistemi kurulmalı:

// Pakete uygulanabilecek kampanyayı belirle (öncelik: en yüksek indirim)
var applicableCampaigns = currentCampaigns
    .Where(c => campaignPackageIds[c.Id].Contains(packageId))
    .OrderByDescending(c => c.DiscountRate ?? 0)
    .ThenByDescending(c => c.DiscountAmount ?? 0)
    .FirstOrDefault();

4. Soft Delete Kullanımı

Kampanyalar soft delete ile silinir. Geçmiş kampanya kayıtları korunur.

5. Audit Trail

Her işlemde CreatedBy, UpdatedBy, DeletedBy alanları otomatik güncellenir.

6. İş Kuralları

  • Bir kampanya birden fazla pakete uygulanabilir
  • Bir paket aynı anda birden fazla kampanyada olabilir (öncelik sistemi gerekir)
  • Kampanya tarihleri geçtikten sonra otomatik devre dışı bırakılmalı
  • GetCurrentCampaignsAsync metodu sadece geçerli kampanyaları döner
  • Kampanya indirimleri teklif oluşturulurken otomatik uygulanmalı

💡 İpuçları ve Püf Noktaları

1. Otomatik Kampanya Uygulama

Teklif oluştururken kampanyaları otomatik kontrol edin:

public async Task<decimal> CalculatePriceWithCampaign(int packageId, decimal originalPrice)
{
    var currentCampaigns = await _campaignService.GetCurrentCampaignsAsync();
    
    foreach (var campaign in currentCampaigns)
    {
        var isInCampaign = await _campaignPackageService.IsPackageInCampaignAsync(
            campaign.Id, packageId);
        
        if (isInCampaign)
        {
            if (campaign.DiscountRate.HasValue)
                return originalPrice * (1 - campaign.DiscountRate.Value / 100);
            
            if (campaign.DiscountAmount.HasValue)
                return originalPrice - campaign.DiscountAmount.Value;
        }
    }
    
    return originalPrice;
}

3. Cache Kullanımı

Geçerli kampanyaları cache'leyerek performans artırın:

private IMemoryCache _cache;

public async Task<List<Campaign>> GetCurrentCampaignsCached()
{
    const string cacheKey = "current_campaigns";
    
    if (!_cache.TryGetValue(cacheKey, out List<Campaign> campaigns))
    {
        campaigns = await _campaignService.GetCurrentCampaignsAsync();
        
        var cacheOptions = new MemoryCacheEntryOptions()
            .SetAbsoluteExpiration(TimeSpan.FromHours(1));
        
        _cache.Set(cacheKey, campaigns, cacheOptions);
    }
    
    return campaigns;
}

4. Background Job - Kampanya Aktivasyonu

Planlı kampanyaları otomatik başlatın:

// Hangfire veya benzer bir background job kütüphanesi ile
public async Task ActivateScheduledCampaigns()
{
    var now = DateTime.UtcNow;
    
    // Başlama tarihi gelmiş ama pasif olan kampanyalar
    var campaignsToActivate = await _context.Campaigns
        .Where(c => !c.IsActive && 
                    c.StartDate <= now && 
                    c.EndDate >= now)
        .ToListAsync();
    
    foreach (var campaign in campaignsToActivate)
    {
        campaign.IsActive = true;
    }
    
    await _context.SaveChangesAsync();
}
