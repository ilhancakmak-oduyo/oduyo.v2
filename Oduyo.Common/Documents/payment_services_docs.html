💳 IPaymentService

💡 Ödeme Nedir? Müşterilerden alınan ödemelerin kaydedildiği ve takip edildiği sistemdir. Sanal POS entegrasyonları ile çalışır ve ödeme durumları (beklemede, tamamlandı, başarısız, iade) takip edilir.

PaymentStatus Enum Değerleri

Değer Açıklama
Pending (1) Beklemede - Ödeme başlatıldı
Completed (2) Tamamlandı - Ödeme başarılı
Failed (3) Başarısız - Ödeme gerçekleşmedi
Refunded (4) İade Edildi - Ödeme iade edildi

Ödeme Oluşturma - CreatePaymentAsync

Web API Controller

[HttpPost]
public async Task<IActionResult> CreatePayment([FromBody] CreatePaymentDto dto)
{
    var currentUserId = int.Parse(User.FindFirst(ClaimTypes.NameIdentifier).Value);
    
    var payment = await _paymentService.CreatePaymentAsync(dto, currentUserId);
    return Ok(new { message = "Ödeme oluşturuldu", data = payment });
}

Blazor Component

@inject IPaymentService PaymentService

private async Task CreatePayment()
{
    var dto = new CreatePaymentDto
    {
        CompanyId = companyId,
        OfferId = offerId,
        Amount = totalAmount,
        CurrencyId = currencyId,
        VirtualPOSId = selectedPOSId
    };
    
    var payment = await PaymentService.CreatePaymentAsync(dto, currentUserId);
    NavigationManager.NavigateTo($"/payment/{payment.Id}");
}

WinForms

private async void btnCreatePayment_Click(object sender, EventArgs e)

}

🏦 IVirtualPOSService

💡 Sanal POS: Online ödeme almak için kullanılan sanal pos entegrasyonlarının yönetildiği sistemdir.

Sanal POS Oluşturma - CreateVirtualPOSAsync

var dto = new CreateVirtualPOSDto
{
    Name = "İş Bankası Sanal POS",
    MerchantId = "MERCHANT123",
    ApiKey = "api_key",
    ApiSecret = "api_secret"
};

var pos = await _virtualPOSService.CreateVirtualPOSAsync(dto, currentUserId);

Sanal POS Listeleme

var allPOS = await _virtualPOSService.GetAllVirtualPOSAsync();

📋 Servis Özet Tablosu

Servis Metod Açıklama Dönüş Tipi
IPaymentService CreatePaymentAsync Ödeme oluşturur Task<Payment>
CompletePaymentAsync Ödemeyi tamamlar Task<bool>
FailPaymentAsync Ödemeyi başarısız işaretler Task<bool>
RefundPaymentAsync Ödemeyi iade eder Task<bool>
GetPaymentByIdAsync ID ile ödeme getirir Task<Payment>
GetCompanyPaymentsAsync Firma ödemelerini getirir Task<List<Payment>>
GetPaymentsByStatusAsync Duruma göre ödemeler Task<List<Payment>>
GetPaymentsByOfferAsync Teklif ödemeleri Task<List<Payment>>
GetCompanyTotalPaymentsAsync Toplam ödeme tutarı Task<decimal>
IVirtualPOSService CreateVirtualPOSAsync Sanal POS oluşturur Task<VirtualPOS>
UpdateVirtualPOSAsync Sanal POS günceller Task<VirtualPOS>
DeleteVirtualPOSAsync Sanal POS siler Task<bool>
GetVirtualPOSByIdAsync ID ile sanal POS Task<VirtualPOS>
GetAllVirtualPOSAsync Tüm sanal POSlar Task<List<VirtualPOS>>
GetActiveVirtualPOSAsync Aktif sanal POSlar Task<List<VirtualPOS>>
